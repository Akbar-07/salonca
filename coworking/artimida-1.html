Описание

Бьюти-коворкинг ARTIMIDA предоставляет в аренду по часам/дням: - три полностью оборудованных кабинета с мокрой точкой - два полностью оборудованных кабинета без мокрой точки. В кабинетах есть все необходимое для вашей работы и комфортного времяпрепровождения ваших клиентов: - Кушетка регулируемая - Косметологический столик- Регулируемый стул - Мокрая точка - Кольцевая лампа - Вешалка для одежды- Кондиционер. Наши сервисы: - Шаговая доступность от метро Савеловская, Менделеевская - Доступные цены, удобная система лояльности - Бесплатная парковка - Шкафчики для хранения - Холодильная камера для ваших препаратов - Администратор - Продажа расходных материалов - Чай/кофе- График работы: ежедневно, 10.00-22.00. Наш Instagram artimida_coworking. Мы будем рады Вас видеть!
